# MyLaya
MyLaya
初学laya，LAYAAir + FGUI入门项目

项目结构:
整个是一个laya空项目结构
src/UI      --> fairygui 生成的代码
UIProject   --> fairygui 原始项目

已做部分：
页面适配和点击显示

启动项目:
方式1:
用layaide打开.laya文件 ==> 直接运行
方式2:
vscode:
需要配置launch.json ==> "runtimeExecutable": "D://Program Files//Chrome//Chrome.exe",
npm install
ctrl+F5


如果想在vscode中编译laya：
请打开tools文件夹
最后在vscode中ctrl+shift+b ==> build 任务运行 ==> F5运行即可
